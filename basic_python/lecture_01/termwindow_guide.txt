code file_name.py : creates a new filepython file_name.py : runs the filecd file_path : changes the directory of the terminalclear : clears the terminal windowls : list all the files in the directorycp old_file.py new_file.py : copies the contents of the old file to the newly created filerm old_file.py : deletes the old filemkdir folder : creates a folderrmdir : removes the foldermv file_name.py .. : this makes the folder move towards the parent folderThe '..' refers to the parent folder, but to travel one branch at a time, to move directly to the parent folder, we need to write 'cd' in the terminal window
